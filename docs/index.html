Daily Feed

Subscribe or unsubscribe from the morning brief. Requests are routed through GitHub and processed by Actions, so the list stays versioned and simple.

Subscription Console

Manage your delivery

Enter your email, then choose whether you want to subscribe or unsubscribe. GitHub will open a prefilled request for you and the automation will update subscribers.txt.

How It Works

Issue-driven updates

Each button opens a GitHub issue request. A workflow reads the request, updates the subscriber file, commits the change, and closes the issue automatically.

Note

GitHub sign-in required

Because everything runs inside GitHub, users need to be signed into GitHub to submit the request issue.
